Table Tennis Tournament
========================

A web app to manage the an epic table tournament held at Makers Academy during the course.

Features
--------
- Player sign up
- Game tracking
- Score tracking
- Group stage tables
- Semi/Quarter/Final match tracking 

Programs and Technologies
-------------------------
- Sublime Text 3
- Ruby
- Rspec
- Capybara
- Sinatra
- HTML5/CSS3
- Postgresql
